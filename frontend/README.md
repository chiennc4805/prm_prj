# My F-School Web

Giao diện ReactJS dành cho Admin và Giáo viên.

## Chạy development

Backend phải chạy tại `http://localhost:8080`.

```bash
npm install
npm run dev
```

Mở địa chỉ Vite hiển thị trong terminal (mặc định `http://localhost:5173`).

## Tài khoản mẫu

- Admin: `0900000001` / `123456`
- GVCN: `0900000002` / `123456`
- Giáo viên bộ môn: `0900000003` / `123456`

Nếu Backend chạy ở URL khác, copy `.env.example` thành `.env` và sửa `VITE_API_URL`.

# FPT Schools UI Drafts

Bộ phác thảo gồm:

- 15 màn hình web dành cho Admin và Giáo viên.
- 15 màn hình mobile dành cho Học sinh và Phụ huynh.
- Mỗi màn hình có cả định dạng SVG và PNG, kích thước 1024 × 1024.
- Mở `index.html` để xem toàn bộ gallery.

Quy tắc thiết kế: nền sáng, card trắng hoặc pastel nhẹ, đường viền xám, typography rõ và màu cam làm điểm nhấn chính. Không thêm watermark hoặc nội dung không tồn tại trên giao diện thật.
